# letsswap

Letsswap is a java webapplication offer platform with it you can swap digital things like images or texts with other user.s

# used technologies

* java
* rest
* bootstrap
* mongodb
* maven


# project structure

* letsswap-web
* letsswap-rest
* letsswap-mongo


# how to build

mvn build

# how to deploy

* cloudfoundry: cf push
